# Credentials (secrets locaux)

Placez ici **uniquement** le fichier JSON du **compte de service Google** utilisé pour Search Console.

- **Nom attendu :** `google-service-account.json` (aligné avec `GOOGLE_APPLICATION_CREDENTIALS` dans `.env`)
- **Ne commitez pas** ce fichier : il est ignoré par Git (voir `.gitignore`).

Les clés **Serper**, **OpenAI**, **Anthropic** se mettent dans le fichier **`.env`** à la racine du projet, pas dans ce dossier.
